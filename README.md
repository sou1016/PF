
# ☆サイト名   
### clothesgram  

# ☆サイト概要    
## ・サイトテーマ  
### 好きな服を検索したり、  
投稿したりできるファッション系のコミュニティサイト。 

## ☆・テーマを選んだ理由    
### 服や靴が好きなのでいろいろなサイトでよく見ているのですが、    
服や靴だけに特化したサイトがあればもっと簡単に欲しい服のブランドが知れたり、  
欲しい服が見つかるのではないかと考えこのテーマにしました。  

## ☆・ターゲットユーザー  
### 服や靴などが好きな人   

## ☆・主な利用シーン  
### 様々な系統のファッションアイテムを検索したり、   
自分の気に入っているファッションアイテムの投稿をする。
